# Ewwon Coco Project Documentation

Selamat datang di dokumentasi resmi proyek **Ewwon Coco Super App**. Folder ini berisi panduan lengkap untuk memahami, mengoperasikan, dan mengembangkan platform lebih lanjut.

## 📁 Daftar Dokumen

1. **[HANDOFF.md](./HANDOFF.md)**
   Berisi ringkasan seluruh fitur yang telah dibangun, detail teknis implementasi (Backend/Frontend), dan instruksi operasional untuk menjalankan sistem secara penuh.

2. **[TEST_ACCOUNTS.md](./TEST_ACCOUNTS.md)**
   Daftar kredensial login untuk pengujian di berbagai level hak akses (Super Admin, Merchant Admin, Kasir, dan Customer).

## 🚀 Gambaran Umum Sistem
Ewwon Coco adalah solusi terintegrasi untuk bisnis F&B kelapa muda yang mencakup:
- **Customer Portal**: Pemesanan online dengan real-time tracking kurir.
- **Merchant Panel**: Manajemen produk, stok, dan kampanye marketing (Referral/Cashback).
- **POS System**: Aplikasi kasir untuk transaksi outlet fisik.
- **Super Admin**: Kontrol sistem global dan analitik multi-merchant.

## 🛠 Prasyarat Teknis
- PHP 8.2+ & MySQL
- Node.js (untuk kompilasi React/Tailwind 4)
- Akun Pusher (untuk fitur Real-time)
- Akun Firebase (untuk Push Notifications)
